高校专项计划
是国家为更好地促进教育公平、让更多的农村学生上大学而出台的一项优惠政策，主要招收边远、贫困、民族等地区县(含县级市)以下高中勤奋好学、成绩优良的农村学生，具体实施区域由有关 省(区、市)确定。

 报考学生需同时具备下列三项条件：
 符合2022年统一高考报名条件；
 本人及父母或法定监护人户籍地在各省实施区域的农村，本人具有当地连续3年以上户籍；
 本人具有户籍所在地高中连续3年学籍并实际就读。
